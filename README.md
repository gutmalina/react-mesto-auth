# Проектная работа по курсу "Веб-разработчик" от Яндекса.Практикум

## ***Mesto Russia - социальная сеть***

Проект создан в рамках учебного курса Яндекс.Практикум и посвящен созданию сервиса Mesto - это социальная сеть, где пользователи могут зарегистрироваться, авторизоваться, изменить свои данные (имя, профессия, аватар), и обмениваться фотографиями, добавлять, удалять изображения, а так же отмечать лайками. Вёрстка адаптивная 320px-1280px
